# Conversor de Monedas
Mi primer proyecto de Back end con Java.

Este proyecto es un **Conversor de Monedas** desarrollado en **Java** utilizando **IntelliJ IDEA**. Permite convertir entre diferentes divisas según una tasa de cambio.

## Características

- Conversión entre diferentes monedas (Dolar, Real brasilero, peso colombiano etc.).
- Uso de api
- Validación de entradas para evitar errores.

## Tecnologías

- **Java**: Lenguaje de programación principal.
- **IntelliJ IDEA**: Entorno de desarrollo utilizado.
